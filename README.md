# Alpha Omega Staking

Simple Solidity staking farm with deposit and reward token contracts.

## Overview

This repository contains a small token farming system where users stake one token and receive rewards in another. It is structured as a compact DeFi learning project.

## Contracts

- `AlphaToken.sol`: stake token
- `OmegaToken.sol`: reward token
- `TokenFarm.sol`: staking and reward distribution contract

## Usage

Compile and deploy the contracts with Remix or another Solidity-compatible toolchain.
